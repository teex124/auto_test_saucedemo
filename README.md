Saucedemo Autotests API

Автоматизированный тестовый проект для проверки веб-приложения Saucedemo. Покрывает основные пользовательские сценарии: авторизация, работа с корзиной, оформление заказа и проверка UI.

Tech Stack:

Framework: Playwright (async)

Language: Python 3.14

Test Runner: Pytest

Architecture: Page Object Model (POM)
